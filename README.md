# React Native Mediterranean Restaurant App

A  **React Native app** for a Mediterranean restaurant, featuring **navigation**,**Local Storage**, **API integration**, and a **three-tab bottom navigation** setup.

---

## Features

- **Welcome Screen** – Initial greeting screen with restaurant branding.
- **Onboarding Screen** -Initial onbaording Screen with restrunat branding 
- **Menu Tab** – Displays the list of menu items fetched from an API.
- **Profile Screen** - Edit and adding the profile of user, adding the image and deleting images.
- **Subscribe Tab** – Allows users to subscribe for updates/offers.
- **Navigation** – Implemented with React Navigation using **Bottom Tabs** and **Stack Navigation** where necessary.
- **API Integration** – Menu data fetched from backend APIs.
- **Add to Cart ** -add to cart screen 
- 
---

## Screens

1. **Welcome Screen** – Landing page when the app opens.
2.  **OnBoarding Screen**- OnBoarding screen contain the login and SignUp screen.
3. **Menu Screen** – Shows all menu items with details.  
4. **Subscribe Screen** – Users can enter details to subscribe for newsletters or offers.

---

## Tech Stack

- **React Native** :-components, Screens, Ui Storage  
- **React Navigation** (Stack & Bottom Tabs)
- **AsyncStorage** (FOr local storage)
- **Axios / Fetch API/Rest Api ** for backend requests
